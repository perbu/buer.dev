CLI Tools

Small, focused command-line tools. All written in Go.

git2llm

Generates a text representation of a git repository's tree and file contents, formatted for LLM consumption. Filters out binary files, dotfiles, and secrets automatically. Supports token counting and custom ignore patterns via .llmignore.

github.com/perbu/git2llm

reviewer

AI code review tool. Pipes a repository through git2llm and sends it to Google Gemini for review. Streams markdown output. Can focus on specific areas like security, performance, or error handling.

github.com/perbu/reviewer

describe

Reads your staged git changes and sends them to an LLM to generate a commit message. Works with a local Ollama install out of the box, or with OpenRouter and vLLM as backends.

github.com/perbu/describe

gosummarize

Scans Go source code and extracts all exported declarations — functions, types, methods, constants, variables — along with their doc comments. Produces a compact summary of a package's public API suitable for LLM context.

github.com/perbu/gosummarize

bump

Reads git tags to find the current version, increments it, updates .version files in the repo, commits the change, and creates a new tag. Handles major, minor, and patch bumps.

github.com/perbu/bump

calvin

Checks Google Calendar from the terminal. Give it a username and optional date and it shows events with times and attendees. Supports relative dates like “tomorrow” and “next monday”.

github.com/perbu/calvin

blamestats

Analyzes git repositories for code ownership and churn statistics. Shows lines of code per author with age breakdown, and tracks how much of each author's code has been modified or deleted over time.

github.com/perbu/blamestats
